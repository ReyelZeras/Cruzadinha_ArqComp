Cruzadinha

1- Ele é o principal item de hardware do computador.

2- É um circuito digital que realiza operações de adição e booleana AND.

3- É a memória dentro da própria CPU que armazena n bits.

4- O processador armazena os dados com os quais está lidando.

5- Os dados são gravados nelas apenas uma vez.

6- A capacidade de permitir que dados sejam regravados no dispositivo.

7- O processo de gravação (e regravação) é muito mais rápido.

8- Tem como função armazenar grandes quantidades de informações.

9- Permite que certos dispositivos de hardware num computador acessem a memória do sistema para leitura e escrita independentemente da CPU.

10- Tambem chamado de slave select.

11- É a via elétrica usada para transferir dados entre componentes de um computador.

12- É usado para transferir dados entre dispositivos.

13- CPU de uso Intermediário da intel.

14- CPU de uso Avançado da intel.

15- Processadores com 2 núcleos.

16- Processadores com 4 ou mais núcleos.
